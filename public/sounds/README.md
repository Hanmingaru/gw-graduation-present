# Sound effects

Drop MP3 files here with these exact names. The app plays them automatically;
if a file is missing, playback fails silently (no error), so the machine still works.

| Filename         | When it plays                                  | Suggested search terms        |
| ---------------- | ---------------------------------------------- | ----------------------------- |
| `lever.mp3`      | Once, when CLICK ME! is pressed                | "slot lever", "ka-chunk click"|
| `spin-loop.mp3`  | Loops while the reels are spinning             | "reel spin", "whirr loop"     |
| `reel-stop.mp3`  | Once per reel as each of the 3 reels lands     | "tick", "clunk", "reel stop"  |
| `win.mp3`        | (not wired yet — reserved for win detection)   | "slot win", "jackpot jingle"  |

## Where to download (free, no attribution required)

- https://pixabay.com/sound-effects/  (recommended)
- https://mixkit.co/free-sound-effects/game/

Prefer short MP3 clips. Keep `spin-loop.mp3` seamlessly loopable (~1–2 s).
